Dog Spotter

This is a social media app that utilizes Firebase Database to share dog photos with friends. It has features like photo sharing, gps location, commenting, posting, user accounts, and more. I created it on my own. I did the artwork myself using Sketch. It is written in Swift for iOS. It is currently in beta, and it will be ready for release this winter.

BreakBoard

This is a macro-enabled Excel spreadsheet utilizing Visual Basic for Applications. The workbook has some custom controls that do time math, formatting, printing, and some other features. This spreadsheet is used at Whole Foods Market in the Portland metropolitan area. I plan on developing a web version of this tool using HTML, CSS, Javascript, and Firebase as a database.
